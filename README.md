SUMMARY
-------

Ext JS

A Drupal module wrapper for the Sencha ExtJS Library.
Only supports ExtJS versions 4.2.0 and up.

REQUIREMENTS
------------

  * [Ext JS 4.2.0 or greater](http://www.sencha.com/)

INSTALLATION
------------

### Install Dependancies

[Download Ext JS](http://www.sencha.com/) Version 4.2.0 or greater and
place in sites/all/libraries/

Change the folder name to "extjs".

CONFIGURATION
-------------

The module can be configured by going to:
http://yoursite.com/admin/islandora/extjs
